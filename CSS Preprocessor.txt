CSS Preprocessor:
 It is a CSS extension that allows to use feature like variables, nesting, imports, mixins, inheritance, etc.
Advntages==>
+->Reusable code snippets can be created. 
+-> The number of errors can be reduced, and also codes can be bloated(Structured).


SASS(Sassy cascading style sheets):

@import=>allows you to use one style sheet in another style sheet.

@mixin=>Helps you to create group off css declaration code that is to be reused thorughout the website.

@includes=>Its let you to use the mixin.

@extend =>Share set of properties from one selector to another.(Inheritance)

@use=>Include code from another file.It allows you to only import what you need from a module whereas @import
      imports entire file and help to avoid naming conflicts.

@function=> Function @function name(){
                                      @return result;
                                      }
@forward=>When group of style sheets have to be reused across multiple style-sheets ,we can merge all of them into a single entry point using @forward and make use of @use to consume in other style-sheets 

@at-root=>It is used in advanced nested with selector functions.It emits everything within it at the root of the doc instead of using normal nesting.
                    example: In scss.file div{
                                      margin:0px;
                                  @at-root.colum{ baccolor:black;
                                                         }  
                                                           }
                             In css.file div{margin:0px}

                                  .colum{ baccolor:black}
